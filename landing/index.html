Analyze data at the speed of thought. Privately.

DataLens is an open-source BI platform where your data never leaves the browser. Analytical SQL runs on-device via DuckDB-WASM. AI features talk to your local Ollama. No cloud uploads, no vendor lock-in, zero telemetry by default.

A single workflow from file drop to polished dashboard — all of it running inside the browser tab you already have open.

DuckDB in your browser

Analytical SQL on millions of rows with zero setup. Load files, query instantly, and keep the whole runtime on-device.

Local AI assistant

Natural language to SQL via Ollama, with prompts and data staying local instead of flowing through a cloud API.

Beautiful charts

40+ interactive chart types with exports, drill-downs, and polished visuals for dashboards, reports, and one-off analysis.

Data pipelines

Compose transforms visually, save reusable recipes, and turn repetitive cleanup into a repeatable browser-side workflow.

ML workflows

Regression, clustering, forecasting, anomaly detection, and more — without leaving the same privacy-first analytics surface.

Privacy first

No telemetry, no tracking, and no vendor lock-in. Run it locally, inspect the code, or self-host it on your own terms.

🛡️

Your data never leaves the browser

Analytical queries execute inside DuckDB-WASM on your machine. AI calls hit a local Ollama endpoint at localhost:11434. Persistence (saved dashboards, auth) stays on infrastructure that you control, not ours.

Up and running in one command

The full app plus local Ollama detection in under a minute. Everything the demo ships with is real — no mocked backend.

Docker (recommended) 

# Clone, then one command.
$ git clone https://github.com/Aandrew-Kl/DataLens
$ cd DataLens
$ docker-compose up

# Open http://localhost:3000

Local dev 

# Frontend only — enough to try it out.
$ npm install
$ npm run dev

# Optional: enable local AI.
$ ollama pull llama3.2
# DataLens auto-detects Ollama on :11434.
