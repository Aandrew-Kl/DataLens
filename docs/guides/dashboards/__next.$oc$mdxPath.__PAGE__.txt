# Charts

## 40+ Chart Types

DataLens ships with more than forty chart implementations, including:

- bar, stacked bar, line, stepped line, area, combo, and donut charts
- bubble, box plot, violin, dot plot, lollipop, and scatter variants
- treemap, sunburst, funnel, sankey, heatmap, and network views
- radar, polar, gauge, candlestick, histogram, and 3D scatter charts

## Chart Builder UI

The chart builder lets you:

- pick a chart family
- map fields to axes and dimensions
- choose aggregation rules
- preview the generated result against the active DuckDB query

## Axes, Colors, and Aggregations

Typical configuration points include:

- X and Y axis field mapping
- grouping or series breakdowns
- aggregate functions such as `COUNT`, `SUM`, `AVG`, `MIN`, and `MAX`
- palette and style choices tuned for dark and light themes

## Export

Depending on the chart type, you can export:

- `PNG`
- `SVG`
- `CSV`

Some export surfaces also generate share URLs or watermark-ready image output.
