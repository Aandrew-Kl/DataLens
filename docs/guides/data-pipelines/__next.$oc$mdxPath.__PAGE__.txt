# Data Pipelines

## Visual Pipeline Builder

The pipeline builders in DataLens let you assemble transformations without leaving the browser. They are best used when you want repeatable data cleanup before charting, dashboarding, or model training.

## Common Transform Steps

Typical steps include:

- filter rows
- join datasets
- aggregate grouped metrics
- sort and derive fields
- reshape data for pivot-style analysis downstream

The repository also includes focused tools for window functions, grouping, sampling, diffing, and other data-preparation tasks.

## Reusable Recipes

When a flow becomes useful beyond one session, store it as a recipe so teammates can rerun the same preparation logic against refreshed data.

## Scheduled Refresh

Saved recipes are the right unit of repeatability.
