# Getting Started

![Sample datasets gallery](/images/screenshots/05-sample-gallery.png)
*Load a sample dataset and start exploring in seconds.*

The fastest path into DataLens is:

1. Start the stack with Docker Compose.
2. Load a CSV, JSON, or Excel file.
3. Run a DuckDB query in the browser.
4. Turn the result into a chart or dashboard.
5. Enable Ollama if you want natural-language workflows.
