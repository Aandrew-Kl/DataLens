# First Query

This walkthrough takes you from raw file upload to a saved dashboard tile.

## 1. Upload a Dataset

Drop a `CSV`, `JSON`, or `Excel` file into DataLens. The file is parsed locally, profiled, and loaded into DuckDB-WASM so you can start querying immediately.

![Dataset upload placeholder](/images/upload-dataset-placeholder.svg)

## 2. Write Your First SQL Query

Open the SQL workspace and start with a simple preview:

```sql
SELECT *
FROM "your_table"
LIMIT 25;
```

Then move to a grouped query:

```sql
SELECT region, SUM(revenue) AS total_revenue
FROM "your_table"
GROUP BY region
ORDER BY total_revenue DESC;
```

![SQL query placeholder](/images/run-query-placeholder.svg)

## 3. Create Your First Chart

Send the query output to the chart builder and map:

- a category to the X-axis
- an aggregated metric to the Y-axis
- optional color or grouping fields for multiple series

Bar, line, pie, treemap, heatmap, and many other views are available once the result set is ready.

![Chart builder placeholder](/images/chart-builder-placeholder.svg)

## 4. Save It to a Dashboard

After the chart looks right, add it to a dashboard, position it in the layout, and save the dashboard configuration for the active dataset.

![Dashboard save placeholder](/images/dashboard-save-placeholder.svg)

## 5. Iterate

Once the first chart is saved, the usual next steps are:

- refine the SQL and re-run it
- compare a second visualization
- add filters or drill-down views
- export the result as CSV, PNG, or SVG
