# Dashboards

## Create a Dashboard

Start from an existing chart or open the dashboard builder directly for the active dataset. Add widgets, give the dashboard a clear title, and save the configuration once the first layout is in place.

## Drag Charts into Position

Dashboards are designed as a responsive canvas, so you can place charts, KPI cards, text blocks, and supporting widgets where they read best for the audience.

## Filters and Drill-Down

Use dashboards when you need to move beyond a single visualization:

- compare multiple metrics side by side
- add filters to narrow the visible slice of data
- open drill-down experiences from charts, pivots, and deeper exploration views

## Sharing Dashboards

DataLens supports a few sharing patterns depending on the workflow:

- export a dashboard as standalone HTML
- copy shareable state URLs from the current analysis context
- hand off the underlying saved queries or chart configs to another contributor
