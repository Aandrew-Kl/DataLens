# AI Assistant

## Natural-Language Queries

Ask questions in plain English such as:

- "Show revenue by region"
- "Which customers have the highest churn risk?"
- "Summarize sentiment by support channel"

DataLens sends the prompt plus schema context to its AI routes and returns generated SQL with an explanation.

## Suggested Questions

The assistant can surface question suggestions based on the current dataset so new users have a fast starting point instead of writing prompts from scratch.

## SQL Explanation

When a query already exists, the explain flow breaks it into:

- what the query is doing
- which tables and columns matter
- how filters, joins, or aggregates shape the result

## Auto-Dashboard Generation

The AI suggestion routes can generate dashboard-style recommendations from schema shape, row counts, and query intent. When Ollama is not available, DataLens falls back to deterministic dashboard suggestions where possible.
