# FAQ

## Is my data sent to the cloud?

No, not by default. The core analytics loop runs locally with DuckDB-WASM in the browser. If you enable AI features, the intended default is local Ollama rather than a hosted provider.

## Do I need OpenAI?

No. OpenAI is optional. DataLens is designed to work without a hosted LLM and can use Ollama locally for natural-language workflows.

## What file sizes are supported?

The backend defaults to `50 MiB`, which is about `52 MB` in decimal terms. You can change the limit with `MAX_UPLOAD_SIZE` if your deployment needs larger uploads.

## Which file formats can I upload?

The frontend workspace is built around CSV, JSON, and Excel ingestion for local analysis.

## Can I self-host?

Yes. The repository includes Docker assets and a compose-based local stack for the frontend, backend, and PostgreSQL.
