# DataLens

DataLens is privacy-first AI data analytics.

It combines DuckDB-WASM in the browser, a Next.js workspace for exploration, and FastAPI services for authentication, persistence, machine learning, and optional local AI assistance. The result is a local-first analytics stack that feels fast without forcing every workflow through a remote warehouse.

## Tech Stack Highlights

- Browser-first analytics with DuckDB-WASM for local profiling, previews, and ad hoc SQL.
- Next.js 16, React 19, TypeScript, and Tailwind CSS v4 for the interactive workspace.
- FastAPI, PostgreSQL, JWT auth, and WebSocket streaming for persistence and backend workflows.
- Optional Ollama-backed AI for natural-language querying, explanations, and dashboard suggestions.
