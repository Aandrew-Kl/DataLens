# Frontend

The frontend is a Next.js 16 application built with React 19, TypeScript, and Tailwind CSS v4.

## Core Responsibilities

- ingest local CSV, JSON, and Excel files
- manage the analytics workspace and feature panels
- run DuckDB-WASM queries in browser memory
- render charts, pivot views, dashboards, and guided analysis tools
- orchestrate AI-assisted actions from the UI

## Important Frontend Building Blocks

- `src/app` for route entrypoints and layouts
- `src/components` for feature UIs grouped by domain
- `src/lib/duckdb` for browser-side SQL execution
- `src/stores` for Zustand-backed state such as datasets, query history, UI state, and bookmarks

## Frontend Design Principle

Keep the exploratory loop close to the user: parse locally, query locally, and only cross the network when the workflow actually needs backend behavior.
